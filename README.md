# Teste Morada
> ## Requisitos

  - [Imóveis](./requirements/properties.md)

> ## Instruções para execução:
 
### Clonar o repositório:

```bash
$ git clone github.com/salobato/morada.git
```
  
## Instalar dependências:

```bash
$ yarn install
```

## Rodando a aplicação

```bash
# build
$ yarn run build

# desenvolvimento
$ yarn run dev

# modo produção
$ yarn run start
```

## Testes

```bash
# testes unitários
$ yarn run test
```
